# 开通APM2.0<a name="apm_07_0002"></a>

1.  请[注册华为云帐号](https://support.huaweicloud.com/usermanual-account/account_id_001.html)并完成[实名认证](https://support.huaweicloud.com/usermanual-account/zh-cn_topic_0071343161.html)。
2.  开通APM2.0免费公测版本。
    1.  登录华为云管理控制台。
    2.  单击左侧![](figures/zh-cn_image_0000001144642310.png)，选择“管理与监管 \> 应用性能管理 APM”进入APM服务页面。
    3.  单击右上角“点击体验新版”进入APM2.0服务页面。

        ![](figures/zh-cn_image_0000001088743436.png)

    4.  单击“免费体验应用监控”，开通服务。

3.  自动创建访问密钥。

    APM2.0通过AK/SK进行签名验证，通过授权的帐户才能上报数据。

    1.  登录APM2.0控制台。
    2.  单击页面菜单“系统管理”，选择“访问密钥”页面。
    3.  查看自动创建的访问密钥。

4.  隐私与敏感信息保护声明。

    由于APM会将运维数据内容展示到APM控制台，请您在使用过程中，注意您的隐私及敏感信息数据保护，不建议将隐私或敏感数据上传到APM，必要时请加密保护。
